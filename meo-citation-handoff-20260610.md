## サイテーション登録時の注意

- 住所、電話、会社名はこのファイルの表記に統一する。
- 旧住所や旧電話番号がある媒体は、削除または修正する。
- 行田市は営業所・対応地域として扱い、法務文書や外部サイテーションの主住所にはしない。
- 低品質なリンク集、相互リンク集、被リンク購入、成果保証型の不自然な登録は避ける。
- プロフィール文を媒体ごとに少し変え、同一文章の大量転載にしない。
- リンク先は基本 `https://digitool-lab.com/`。補助金・事例探索の文脈では `https://showroom.digitool-lab.com/` も使う。

## 記録項目

サイテーション作業では、次の項目を残す。

| 項目 | 内容 |
|---|---|
| 媒体名 | 登録・更新した媒体名 |
| URL | 公開プロフィールまたは掲載ページ |
| 反映日 | 作業日 |
| NAP一致 | 会社名、住所、電話が一致しているか |
| リンク先 | 公式HP、showroom、資料DLなど |
| 状態 | 申請中、公開済み、修正待ち、却下 |
| 備考 | 審査、ログイン、本人確認、追加資料など |
