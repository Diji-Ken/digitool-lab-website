課題：アイデア出しが行き詰まる

企業プロフィール

業種 食品卸売業
従業員数 19名
主要取引先 飲食店、小売店約150社
主な課題 新規事業や販促のアイデアが出にくい

アイデア出しの現状

同社では市場環境の変化に対応するため、新しい事業や販促施策のアイデアを常に求めています。しかし、社内でのアイデア出しには限界がありました。

アイデア出しの課題

  • 同じメンバーでの議論：視点が似通っており、新しい発想が出にくい
  • 業界の常識に縛られる：「うちの業界では無理」という思い込み
  • 時間の制約：ブレストに十分な時間を取れない
  • 批判への懸念：突飛なアイデアを出しにくい雰囲気

従来のアイデア出しフロー

  1. 会議を設定（参加者調整に時間がかかる）
  2. ブレインストーミング実施（30分〜1時間）
  3. アイデア整理（30分）
  4. 実現可能性の検討（別日に再会議）

問題点：結局「いつものアイデア」に落ち着きがち

担当者の声（導入前）

「会議でアイデア出しをしても、結局みんな似たようなことを言うんです。『他業界ではこうやってる』みたいな情報があればいいのですが、調べる時間もなくて...」

— 営業企画担当

施策：Geminiでアイデア発想を支援

GeminiをブレストパートナーとしてActivate

Geminiに課題や目的を伝え、多角的な視点からアイデアを提案してもらう仕組みを構築しました。

AI活用アイデア出しフロー

  1. 課題・目的の整理：何についてアイデアが欲しいか明確化
  2. Geminiに問いかけ：複数の視点からアイデアを依頼
  3. アイデアの拡張：出てきたアイデアを深掘り・組み合わせ
  4. 人間が選別・評価：実現可能性を検討
  5. チームで議論：有望なアイデアを会議で深掘り

活用パターン1：多角的な視点からのアイデア出し

異なる立場・視点からアイデアを出してもらいます。

Geminiへの指示例

当社は食品卸会社です。飲食店向けの新しいサービスを検討しています。

以下の5つの視点から、それぞれ3つずつアイデアを提案してください。

1. 飲食店オーナーの視点：何に困っているか
2. 調理スタッフの視点：日常業務で欲しいもの
3. 他業界の成功事例から：異業種で成功しているサービスの応用
4. テクノロジー活用の視点：最新技術で可能になること
5. サステナビリティの視点：環境・社会貢献につながるもの

各アイデアについて、概要と当社で実現する場合のポイントを記載してください。

活用パターン2：既存アイデアの深掘り・拡張

出てきたアイデアをさらに深掘りします。

深掘りの指示例

「サブスクリプション型仕入れ」のアイデアをさらに深掘りしてください。

具体的に教えてほしいこと：
1. ターゲット顧客像（どんな飲食店が使いたがるか）
2. 具体的なサービス内容（価格帯、配送頻度、商品構成）
3. 競合・類似サービスの有無
4. 始めるために必要なこと（最小限で始める場合）
5. 想定される課題とその対策

活用パターン3：アイデアの組み合わせ

複数のアイデアを組み合わせて新しい価値を生み出します。

組み合わせの指示例

以下の2つのアイデアを組み合わせて、新しいサービスアイデアを考えてください。

アイデアA：サブスクリプション型仕入れ
アイデアB：食材ロス削減マッチング

これらを組み合わせることで生まれる新しい価値は何ですか？
具体的なサービス案を3つ提案してください。

活用パターン4：制約条件下でのアイデア出し

現実的な制約を考慮したアイデアを求めます。

制約付きアイデア出しの例

以下の制約条件の中で実現できる新規サービスのアイデアを5つ提案してください。

【制約条件】
・初期投資：100万円以内
・人員：既存スタッフで対応（新規採用なし）
・期間：3ヶ月以内に開始可能
・対象：既存取引先へのサービス拡充

【当社の強み】
・150社との取引関係
・週3回の配送網
・食品の目利き力

活用パターン5：批判的視点からの検証

アイデアに対する批判や懸念を洗い出します。

批判的検証の例

「サブスクリプション型仕入れサービス」のアイデアについて、
批判的な視点から検証してください。

以下の観点からリスクや懸念を洗い出してください：
1. 顧客側のデメリット・不満要因
2. 競合からの攻撃ポイント
3. オペレーション上の課題
4. 収益性の課題
5. このサービスが失敗するシナリオ

それぞれの懸念に対する対策案も併せて提案してください。

定性的な成果

発想力の向上

  • 視野の拡大：他業界の事例を知ることで発想が広がった
  • 固定観念の打破：「うちでは無理」と決めつけなくなった
  • 深掘りの習慣化：アイデアを出して終わりではなく、掘り下げるように

会議の質の向上

  • 準備の質：会議前にAIと壁打ちしてから参加
  • 議論の効率化：ゼロからではなく、叩き台がある状態でスタート
  • 前向きな雰囲気：アイデアが出やすい空気に変化

— 営業企画担当

まとめ：AIで思考の壁を突破する

成功のポイント

  1. 目的の明確化

    「何についてアイデアが欲しいのか」を具体的に伝えることで、有用なアイデアが得られる。

  2. 複数視点の指定

    異なる立場・視点を指定することで、多角的なアイデアが出る。

  3. 深掘り・拡張の習慣化

    出てきたアイデアで終わらず、さらに掘り下げることで実現可能性が高まる。

  4. 人間が最終判断

    AIのアイデアはあくまで叩き台。実行するかどうかは人間が判断。

他業務への応用

  • 商品開発：新商品のコンセプト出し
  • マーケティング施策：販促キャンペーンのアイデア
  • 業務改善：効率化のアイデア出し
  • 課題解決：発生している問題への対策案

効果的なプロンプトのコツ

  1. 背景情報を伝える：業種、規模、課題などのコンテキスト
  2. 制約条件を明示：予算、期間、人員などの制約
  3. 視点を指定：どの立場からのアイデアが欲しいか
  4. 出力形式を指示：箇条書き、表形式など希望を伝える
  5. 追加質問で深掘り：一度で終わらず、対話的に深める
