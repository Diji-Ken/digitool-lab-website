# 被リンク・外部導線候補リスト - 2026-06-10

## 位置づけ

サイテーション登録ではなく、自然な被リンク、外部導線、共同発信、登壇、事例掲載につながる候補を整理する。目的はSEOだけでなく、さいたま市・埼玉県内の中小企業からの相談導線を増やすこと。

## 基本方針

- 公式HP、事例ページ、資料DL、showroomへ戻す導線を作る。
- 有料リンク購入、相互リンク集、低品質ディレクトリ登録はしない。
- 先方に価値がある切り口だけ提案する。例: 中小企業DX事例、社内ポータル要件定義、AI研修、補助金前の要件整理。
- 掲載依頼ではなく、まずは「事例提供」「セミナー共催」「資料提供」「支援者候補登録可否」の順で検討する。

## 初回に出す提案文の型

```
株式会社デジタルツール研究所の松岡です。
さいたま市・埼玉県内の中小企業向けに、社内ポータル、業務システム、生成AI活用研修、DX定着支援を行っています。

貴団体のDX支援情報を拝見し、当社で作成している「中小企業がDX導入前に整理すべき業務・要件」や、社内ポータル/業務システムの事例が、会員企業様や相談者様の参考になる可能性があると考えました。

事例提供、セミナー協力、資料提供、支援者候補登録など、掲載・連携可能な制度があればご教示いただけますでしょうか。
```

## やらないこと

- 被リンク購入
- 自動登録型リンク集
- 低品質な相互リンク
- 実態のない地域ページへの誘導
- 自社HPと同じ記事の丸ごと転載
