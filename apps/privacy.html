プライバシーポリシー

アプリケーション向け - 最終更新日: 2025年1月8日

1. はじめに

株式会社デジタルツール研究所（以下「当社」）は、当社が提供するアプリケーション（以下「本アプリ」）において、ユーザーの皆様のプライバシーを尊重し、個人情報の保護に努めております。

本プライバシーポリシーは、本アプリを通じて収集する情報の種類、その使用方法、および保護方法について説明するものです。

2. 収集する情報

本アプリは、サービス提供のために以下の情報を収集する場合があります：

2.1 アカウント情報

  • Google/YouTubeアカウントの基本プロフィール情報（名前、メールアドレス）
  • OAuth認証に必要なアクセストークン

2.2 サービス利用に必要なデータ

  • YouTube：チャンネル情報、動画メタデータ、アップロード権限
  • Google Workspace：ドキュメント、スプレッドシートへのアクセス権限
  • その他連携サービスに応じた必要最小限のデータ

重要：当社は、サービス提供に必要な最小限の権限のみを要求します。不要なデータへのアクセスは行いません。

3. 情報の使用目的

収集した情報は、以下の目的でのみ使用されます：

  1. 本アプリの機能提供（コンテンツのアップロード、管理、自動化処理）
  2. ユーザー認証およびアカウント管理
  3. サービスの改善および新機能の開発
  4. ユーザーサポートの提供
  5. 利用規約違反の防止および対応

4. 情報の共有

当社は、以下の場合を除き、ユーザーの個人情報を第三者と共有することはありません：

  • ユーザーの明示的な同意がある場合
  • 法令に基づく開示請求があった場合
  • サービス提供に必要な業務委託先（機密保持契約を締結済み）
  • 当社の権利、財産、安全を保護するために必要な場合

なお、連携するサードパーティサービス（Google、YouTube等）には、それぞれのプライバシーポリシーが適用されます。

5. データの保存と保護

当社は、収集したデータを以下の方法で保護します：

  • SSL/TLS暗号化による通信の保護
  • アクセス権限の厳格な管理
  • 定期的なセキュリティ監査の実施
  • 従業員への情報セキュリティ教育

データは、サービス提供に必要な期間のみ保存され、不要になった時点で安全に削除されます。

6. ユーザーの権利

ユーザーは、以下の権利を有します：

  • アクセス権：当社が保有するご自身の個人情報の開示を請求する権利
  • 訂正権：不正確な個人情報の訂正を請求する権利
  • 削除権：個人情報の削除を請求する権利
  • 連携解除：いつでもアプリとの連携を解除する権利

これらの権利を行使される場合は、下記のお問い合わせ先までご連絡ください。

7. Google API サービスの利用

本アプリは、Google APIサービスを利用しています。Google APIから取得した情報の使用および転送は、Google API サービスのユーザーデータポリシー（Limited Use requirements を含む）に準拠します。

8. ポリシーの変更

当社は、本プライバシーポリシーを随時更新する場合があります。重要な変更がある場合は、本アプリまたは当社ウェブサイトにてお知らせします。

本ポリシーの最終更新日は、ページ上部に記載されています。
